## 🕵️ Overview
This is a specialized dashboard designed for real-time monitoring of an "Agent's" vitals and tactical data. It acts as a central hub, receiving live telemetry from a **Galaxy Watch 5** and **iPhone 13**, hosted on a **Raspberry Pi** server.

---

## 🛠️ Tech Stack
* **Backend:** Node.js & Express API
* **Real-time:** Socket.io (WebSockets)
* **Frontend:** Vanilla JS / CSS3 (Cyberpunk/Terminal UI)
* **Hardware:** Raspberry Pi (Server), Galaxy Watch 5 (Sensors)

---

## 🚀 Features
* ❤️ **Live Heart Rate:** Real-time BPM monitoring.
* 📍 **GPS Tracking:** Current tactical coordinates.
* 🚨 **SOS Alert:** Visual emergency notification system.
* 🔋 **Device Status:** Battery and connectivity levels.
* 👣 **Activity:** Live step counter and calorie burn.
